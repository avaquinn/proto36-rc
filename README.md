# PROTO36 Connected RC Car

A Raspberry Pi- and STM32-based RC car platform with low-latency FPV video,
remote browser controls, cellular connectivity, and future autonomous features.

## Current status

- PROTO36 chassis assembled and driving
- Raspberry Pi camera operational
- MediaMTX WebRTC video stream operational
- Python WebSocket command server in development
- STM32 steering and ESC integration planned
